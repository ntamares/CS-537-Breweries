1. JoViewPlus

The JoViewPlus sample is available from the MOJ20\Samples\JoViewPlus folder. It provides a main interface from which a user can perform GIS functions such as adding data layers to the map, querying the data, changing projections, and viewing layout mode. JoViewPlus was built by implementing many of the classes and methods included in the MapObjects API. Many of the new features available in version 2.0 are shown in JoViewPlus. To run JoViewPlus directly, open the StartJoviewPlus file located in MOJ20\Scripts.

2. JoView

The JoView sample is available from the MOJ20\Samples\JoView folder. It is similar to JoviewPlus, except it is built mainly from the version 1.0.1 MapObjects API. It is a good base example without the addition of most of the new features from version 2.0. To run JoView directly, open the StartJoView file located in MOJ20\Scripts.

3. Tutorial

The Tutorial samples are available from the MOJ20\Samples\Tutorial folder. These samples provide demos, descriptions for each of the samples, and source code for the samples—all wrapped into one book style application. They cover a wide range of topics, including how to display images, how to create a MemoryFeature class, information on spatial operations, tips for using coordinate systems, and many others. To open the tutorial directly, navigate to the MOJ20\Scripts folder and open the StartTutorial file.

4. Server samples

The Server samples are available from the MOJ20\Samples\Server folder. Inside the Server folder there is source code, sample AXL files, and other necessary components for running the samples. Within the J2EEServers folder there are separate components and instructions for the individual servers and servlets: JBoss, Sun ONE, Tomcat, WebLogic, ServletExec, and Websphere. There are instructions in the MOJ20\Samples\Server\docs\Introduction_to_MOJ_Server_Samples.html file that give an overview of the server components. That document also contains links to the Javadoc and specific instructions for each of the server/servlet variations. Step-by-step instructions are given to set up JSP Viewers, JSP Servlet Viewers, simple EJB viewers, or more advanced EJB viewers. For further information, refer to the Introduction_to_MOJ_Server_Samples.html file.
