Utilities

There are various utilities included for viewing resources, upgrading 1.x applications, helping deploy applications, and retrieving coordinate system information. The following list gives a brief description of each and specifies where to find more information and access the tools.

Upgrading 1.x Applications Tool

RenameMo can be used as the first step to modify Java source code and property files that currently depend on the MapObjects 1.0 API to work with the MapObjects 2.0 API. RenameMo reads a file directory containing input source and properties files and copies all files found in the input directory to a new file directory with changes that are defined within the renameText method—basically, all references to 'com.esri.mo.' are converted to 'com.esri.mo2.'  For more information, refer to the Readme.txt file in the MOJ20/Utilities/RenameMo folder.  

Coordinate System Tool

The ViewCoordSys utility will extract coordinate system, datum, and projection information from the GeodeticDB.txt file in the esri_mo20res.jar file. Data is retrieved in a table structure format for various attributes. The EPSG IDs are also among the data included. For more information, refer to the Readme.txt file in the MOJ20/Utilities/ViewCoordSys folder.

Deployment Plugin Aid

The DeployPlugin desktop application provides a way to build deployment resources to ensure the MOJava plug-in architecture works within secured Java Virtual Machines, such as an Enterprise Java Beans container. For more details and instructions on how to use this, refer to the Readme.txt file in MOJ20/Utilities/DeployPlugin folder.

Icon Viewer

The IconViewer folder provides a valuable resource listing for MapObjects—Java. The resources included in the esri_mo20.jar are displayed in a package tree structure with descriptions and names for each of the icons. This allows for quick identification of icons that can be used in an application. There is also a small sample showing how to utilize look-and-feel specific icons for use on various platforms. For more information, refer to the index.html file in the MOJ20/Utilities/IconViewer folder.
